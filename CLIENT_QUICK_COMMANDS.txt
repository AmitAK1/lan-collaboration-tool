🔗 CONNECTING:

When the dialog appears:
  1. Enter Server IP (e.g., 192.168.1.100)
  2. Enter your Nickname
  3. Click OK

═══════════════════════════════════════════════════════════════

🔍 FINDING SERVER IP:

Ask the server host to run this command:

  Windows:     ipconfig
  Linux/Mac:   ifconfig  OR  ip addr show

Look for IPv4 Address (e.g., 192.168.x.x or 172.x.x.x)

RUN CLIENT (Linux/Mac):

═══════════════════════════════════════════════════════════════

🐛 COMMON ISSUES:

❌ "pip not found"
   → Add Python to PATH or use: python -m pip install ...

❌ "PyAudio installation failed"
   → Windows: Download from https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
   → Linux: sudo apt-get install portaudio19-dev python3-pyaudio
   → Mac: brew install portaudio && pip3 install pyaudio

❌ "Permission denied"
   → Use: pip install --user [package_name]

❌ "Connection failed"
   → Check if server is running
   → Verify you're on the same network
   → Check firewall settings
   → Confirm correct server IP

═══════════════════════════════════════════════════════════════

📞 PORTS USED:
  • TCP 6543 - Chat & Files
  • UDP 6544 - Audio & Video

Make sure these ports are not blocked by firewall!

═══════════════════════════════════════════════════════════════

✅ VERIFY INSTALLATION:

Windows:
  python -c "import numpy, pyaudio, PIL, mss, cv2; print('OK')"

Linux/Mac:
  python3 -c "import numpy, pyaudio, PIL, mss, cv2; print('OK')"

If you see "OK", you're ready to go! 🎉
